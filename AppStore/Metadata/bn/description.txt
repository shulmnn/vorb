Vorb একটি কীবোর্ড শর্টকাট দিয়ে আপনার ভয়েসকে পাঠ্যে পরিণত করে।

শুরু করতে এবং থামাতে Option–Space টিপুন, অথবা হোল্ড-টু-স্পিক বেছে নিন। Vorb আপনার রেকর্ডিং প্রতিলিপি করে এবং ফলাফলটি ক্লিপবোর্ডে অনুলিপি করে।

ব্যক্তিগত · স্থানীয় হুইসপার
আপনার Mac-এ কোনো API কী ছাড়াই আপনার ভয়েস সম্পূর্ণরূপে পাঠ্যে প্রতিলিপি করুন৷ আপনি মডেল ডাউনলোড ক্লিক করলেই মডেলগুলি ডাউনলোড হয়৷ আপনি একটি বিদ্যমান WhisperKit মডেলও আমদানি করতে পারেন৷

আপনার নিজস্ব প্রদানকারী আনুন
Groq, OpenAI, Deepgram, Mistral, ElevenLabs, একটি স্ব-হোস্টেড সার্ভার, বা অন্য একটি কমপয়েন্ট সংযোগ করুন৷ কীগুলি macOS Keychain-এ থাকে এবং অডিও সরাসরি আপনার নির্বাচিত প্রদানকারীর কাছে যায়৷

ফোকাসড কাজের জন্য নির্মিত
• কনফিগারযোগ্য গ্লোবাল শর্টকাট
• টগল-টু-স্পিক বা হোল্ড-টু-স্পিক
• ন্যূনতম রেকর্ডিং কক্ষ
• ক্লিপবোর্ড ডেলিভারি
• Optional স্থানীয় প্রতিলিপি ইতিহাস
• স্বয়ংক্রিয় ভাষা সনাক্তকরণ
• কোন Vorb অ্যাকাউন্ট, বিশ্লেষণ, বিজ্ঞাপন, বা সদস্যতা নেই

Local Whisper অ্যাপল সিলিকন প্রয়োজন৷ অনলাইন প্রদানকারী এবং মডেল ডাউনলোডের জন্য একটি নেটওয়ার্ক সংযোগ প্রয়োজন।
