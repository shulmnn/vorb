Vorb — це рідна програма для диктування для Mac, яка перетворює мову та голос на текст. Натисніть одну комбінацію клавіш, говоріть природно та перетворюйте свій голосовий ввід на чистий текст, де б ви не писали.

Голосове введення, без набору тексту
Натисніть Option–Space, щоб розпочати та зупинити, або виберіть «Натисни і говори» та утримуйте ярлик під час розмови. Vorb транскрибує ваш запис, копіює результат у буфер обміну та підтримує процес запису в нотатках, повідомленнях, документах, електронній пошті та будь-якій іншій програмі Mac.

Приватний на пристрої Whisper
Запустіть розпізнавання мовлення повністю на своєму Mac за допомогою Local Whisper — API key не потрібен. Моделі Whisper завантажуються лише після натискання «Завантажити модель», а натомість можна імпортувати наявну модель WhisperKit. Місцева транскрипція зберігає аудіо на пристрої.

Принесіть свій власний провайдер STT
Під’єднайте Groq, OpenAI, Deepgram, Mistral, ElevenLabs, сервер Whisper, розміщений на власному хості, або іншу сумісну кінцеву точку перетворення мови в текст. Ваші API key залишаються в macOS Keychain, а звук надходить безпосередньо до вибраного вами постачальника.

Швидке диктування Mac
• Настроюваний глобальний ярлик
• Перемкніть і розмовляйте або натисніть і розмовляйте
• Сфера голосового запису
• Автоматична доставка буфера обміну
• Optional локальна історія розшифровки
• Автоматичне визначення мови
• Немає облікового запису Vorb, аналітики, реклами чи підписки

Vorb робить введення голосом, транскрипцію STT і диктування Whisper рідними для macOS. Для Local Whisper потрібен силікон Apple. Інтернет-провайдери та завантаження моделей потребують підключення до мережі.
