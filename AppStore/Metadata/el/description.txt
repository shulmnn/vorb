Το Vorb είναι μια εγγενής εφαρμογή υπαγόρευσης για το Mac που μετατρέπει την ομιλία και τη φωνή σε κείμενο. Πατήστε μία συντόμευση πληκτρολογίου, μιλήστε φυσικά και μετατρέψτε τη φωνητική σας εισαγωγή σε καθαρό κείμενο όπου κι αν γράφετε.

Φωνητική είσοδος, χωρίς την πληκτρολόγηση
Πατήστε Option–Space για να ξεκινήσετε και να σταματήσετε ή επιλέξτε push-to-talk και κρατήστε πατημένο τη συντόμευση ενώ μιλάτε. Το Vorb μεταγράφει την εγγραφή σας, αντιγράφει το αποτέλεσμα στο πρόχειρο και διατηρεί τη ροή της γραφής σας σε σημειώσεις, μηνύματα, έγγραφα, email και οποιαδήποτε άλλη εφαρμογή Mac.

Ιδιωτική στη συσκευή Whisper
Εκτελέστε την αναγνώριση ομιλίας εξ ολοκλήρου στο Mac με το Local Whisper—δεν απαιτείται API key. Τα μοντέλα Whisper πραγματοποιούνται λήψη μόνο όταν κάνετε κλικ στην επιλογή Λήψη μοντέλου και μπορείτε να εισαγάγετε ένα υπάρχον μοντέλο WhisperKit. Η τοπική μεταγραφή διατηρεί τον ήχο σας στη συσκευή.

Φέρτε τον δικό σας πάροχο STT
Συνδέστε τους Groq, OpenAI, Deepgram, Mistral, ElevenLabs, έναν αυτο-φιλοξενούμενο διακομιστή Whisper ή άλλο συμβατό τελικό σημείο ομιλίας σε κείμενο. Τα API key σας παραμένουν στο macOS Keychain και ο ήχος πηγαίνει απευθείας στον πάροχο που επιλέγετε.

Γρήγορη υπαγόρευση Mac
• Καθολική συντόμευση με δυνατότητα διαμόρφωσης
• Εναλλαγή για ομιλία ή push-to-talk
• Σφαίρα εγγραφής με αντίδραση φωνής
• Αυτόματη παράδοση στο πρόχειρο
• Ιστορικό τοπικής μεταγραφής Optional
• Αυτόματος εντοπισμός γλώσσας
• Δεν υπάρχει λογαριασμός, αναλυτικά στοιχεία, διαφημίσεις ή συνδρομή Vorb

Το Vorb κάνει τη φωνητική πληκτρολόγηση, τη μεταγραφή STT και την υπαγόρευση Whisper να φαίνονται εγγενείς στο macOS. Το Local Whisper απαιτεί πυρίτιο Apple. Οι διαδικτυακοί πάροχοι και οι λήψεις μοντέλων απαιτούν σύνδεση δικτύου.
