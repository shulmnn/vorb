Vorb เปลี่ยนเสียงของคุณให้เป็นข้อความด้วยแป้นพิมพ์ลัดเพียงปุ่มเดียว

กด Option–Space เพื่อเริ่มและหยุด หรือเลือกถือเพื่อพูด Vorb ถอดเสียงการบันทึกของคุณและคัดลอกผลลัพธ์ไปยังคลิปบอร์ด

ส่วนตัว · เสียงกระซิบในท้องถิ่น
ถอดเสียงของคุณเป็นข้อความทั้งหมดบน Mac โดยไม่ต้องใช้คีย์ API โมเดลจะดาวน์โหลดเมื่อคุณคลิกดาวน์โหลดโมเดลเท่านั้น คุณยังสามารถนำเข้าโมเดล WhisperKit ที่มีอยู่ได้

นำผู้ให้บริการของคุณเอง
เชื่อมต่อ Groq, OpenAI, Deepgram, Mistral, ElevenLabs, เซิร์ฟเวอร์ที่โฮสต์ในตัวเอง หรือจุดสิ้นสุดอื่นที่เข้ากันได้ ปุ่มจะอยู่ใน macOS Keychain และเสียงจะถูกส่งไปยังผู้ให้บริการที่คุณเลือกโดยตรง

สร้างขึ้นเพื่อการทำงานที่มุ่งเน้น
•ทางลัดทั่วโลกที่กำหนดค่าได้
• สลับเพื่อพูดหรือถือเพื่อพูด
• ลูกโลกบันทึกน้อยที่สุด
• จัดส่งคลิปบอร์ด
• Optional ประวัติการถอดเสียงในเครื่อง
• การตรวจจับภาษาอัตโนมัติ
• ไม่มีบัญชี Vorb การวิเคราะห์ การโฆษณา หรือการสมัครสมาชิก

Local Whisper ต้องใช้ Apple Silicon ผู้ให้บริการออนไลน์และการดาวน์โหลดโมเดลจำเป็นต้องมีการเชื่อมต่อเครือข่าย
