Vorb เป็นแอปเขียนตามคำบอกแบบดั้งเดิมสำหรับ Mac ที่เปลี่ยนคำพูดและเสียงเป็นข้อความ กดแป้นพิมพ์ลัดเดียว พูดอย่างเป็นธรรมชาติ และเปลี่ยนการป้อนข้อมูลด้วยเสียงของคุณให้เป็นข้อความที่สะอาดตาทุกที่ที่คุณเขียน

ป้อนข้อมูลด้วยเสียงโดยไม่ต้องพิมพ์
กด Option–Space เพื่อเริ่มและหยุด หรือเลือกกดเพื่อพูดและกดทางลัดค้างไว้ขณะพูด Vorb ถอดเสียงการบันทึกของคุณ คัดลอกผลลัพธ์ไปยังคลิปบอร์ด และช่วยให้การเขียนของคุณเคลื่อนไหวในโน้ต ข้อความ เอกสาร อีเมล และแอป Mac อื่นๆ

ส่วนตัวบนอุปกรณ์ Whisper
เรียกใช้การรู้จำเสียงทั้งหมดบน Mac ของคุณด้วย Local Whisper โดยไม่ต้องใช้ API key โมเดล Whisper จะดาวน์โหลดเมื่อคุณคลิกดาวน์โหลดโมเดลเท่านั้น และคุณสามารถนำเข้าโมเดล WhisperKit ที่มีอยู่แทนได้ การถอดเสียงเป็นคำในเครื่องจะเก็บเสียงของคุณไว้ในอุปกรณ์

นำผู้ให้บริการ STT มาเอง
เชื่อมต่อ Groq, OpenAI, Deepgram, Mistral, ElevenLabs ซึ่งเป็นเซิร์ฟเวอร์ Whisper ที่โฮสต์เอง หรือจุดสิ้นสุดคำพูดเป็นข้อความอื่นที่เข้ากันได้ API key ของคุณจะอยู่ใน macOS Keychain และเสียงจะถูกส่งไปยังผู้ให้บริการที่คุณเลือกโดยตรง

การเขียนตามคำบอก Mac ที่รวดเร็ว
•ทางลัดทั่วโลกที่กำหนดค่าได้
• สลับเพื่อพูดหรือกดเพื่อพูด
• ลูกโลกบันทึกปฏิกิริยาด้วยเสียง
• การจัดส่งคลิปบอร์ดอัตโนมัติ
• Optional ประวัติการถอดเสียงในเครื่อง
• การตรวจจับภาษาอัตโนมัติ
• ไม่มีบัญชี Vorb การวิเคราะห์ การโฆษณา หรือการสมัครสมาชิก

Vorb ทำให้การพิมพ์ด้วยเสียง การถอดเสียง STT และการเขียนตามคำบอก Whisper ให้ความรู้สึกเป็นธรรมชาติเหมือน macOS Local Whisper ต้องใช้ Apple Silicon ผู้ให้บริการออนไลน์และการดาวน์โหลดโมเดลจำเป็นต้องมีการเชื่อมต่อเครือข่าย
