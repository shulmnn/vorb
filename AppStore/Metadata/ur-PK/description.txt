Vorb Mac کے لیے ایک مقامی ڈکٹیشن ایپ ہے جو تقریر اور آواز کو متن میں بدل دیتی ہے۔ ایک کی بورڈ شارٹ کٹ دبائیں، قدرتی طور پر بولیں، اور جہاں بھی آپ لکھیں اپنے صوتی ان پٹ کو صاف متن میں تبدیل کریں۔

صوتی ان پٹ، ٹائپنگ کے بغیر
شروع کرنے اور روکنے کے لیے Option–Space دبائیں، یا پش ٹو ٹاک کا انتخاب کریں اور بولتے وقت شارٹ کٹ کو پکڑیں۔ Vorb آپ کی ریکارڈنگ کو نقل کرتا ہے، نتیجہ کو کلپ بورڈ پر کاپی کرتا ہے، اور آپ کے تحریری بہاؤ کو نوٹ، پیغامات، دستاویزات، ای میل، اور کسی دوسرے Mac ایپ میں منتقل کرتا ہے۔

پرائیویٹ آن ڈیوائس Whisper
اسپیچ ریکگنیشن کو مکمل طور پر اپنے Mac پر Local Whisper کے ساتھ چلائیں — API key کی ضرورت نہیں۔ Whisper ماڈل صرف اس وقت ڈاؤن لوڈ ہوتے ہیں جب آپ ڈاؤن لوڈ ماڈل پر کلک کریں، اور آپ اس کے بجائے موجودہ WhisperKit ماڈل درآمد کر سکتے ہیں۔ مقامی ٹرانسکرپشن آپ کے آڈیو کو ڈیوائس پر رکھتی ہے۔

اپنا خود کا STT فراہم کنندہ لائیں۔
Groq, OpenAI, Deepgram, Mistral, ElevenLabs، ایک خود میزبان Whisper سرور، یا کوئی اور مطابقت پذیر اسپیچ ٹو ٹیکسٹ اینڈ پوائنٹ کو مربوط کریں۔ آپ کے API keys macOS Keychain میں رہتے ہیں، اور آڈیو براہ راست آپ کے منتخب کردہ فراہم کنندہ کے پاس جاتا ہے۔

تیز Mac ڈکٹیشن
• قابل ترتیب عالمی شارٹ کٹ
• بولنے کے لیے ٹوگل کریں یا بات کرنے کے لیے دھکیلیں۔
• وائس ری ایکٹیو ریکارڈنگ اورب
• خودکار کلپ بورڈ کی ترسیل
• Optional مقامی نقل کی تاریخ
• خودکار زبان کا پتہ لگانا
• کوئی Vorb اکاؤنٹ، تجزیات، اشتہارات، یا سبسکرپشن نہیں۔

Vorb صوتی ٹائپنگ، STT ٹرانسکرپشن، اور Whisper ڈکٹیشن کو macOS کا مقامی محسوس کرتا ہے۔ Local Whisper ایپل سلکان کی ضرورت ہے۔ آن لائن فراہم کنندگان اور ماڈل ڈاؤن لوڈ کے لیے نیٹ ورک کنکشن کی ضرورت ہوتی ہے۔
