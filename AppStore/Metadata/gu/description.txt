Vorb એક કીબોર્ડ શોર્ટકટ વડે તમારા અવાજને ટેક્સ્ટમાં ફેરવે છે.

શરૂ કરવા અને રોકવા માટે Option–Space દબાવો અથવા હોલ્ડ-ટુ-સ્પીક પસંદ કરો. Vorb તમારા રેકોર્ડિંગને ટ્રાંસ્ક્રાઇબ કરે છે અને પરિણામને ક્લિપબોર્ડ પર કૉપિ કરે છે.

ખાનગી · સ્થાનિક વ્હીસ્પર
API કી વગર તમારા Mac પર તમારા વૉઇસને સંપૂર્ણ રીતે ટેક્સ્ટમાં ટ્રાંસ્ક્રાઇબ કરો. જ્યારે તમે મોડલ ડાઉનલોડ કરો પર ક્લિક કરો ત્યારે જ મોડલ્સ ડાઉનલોડ થાય છે. તમે હાલનું WhisperKit મોડલ પણ આયાત કરી શકો છો.

તમારા પોતાના પ્રદાતાને લાવો
Groq, OpenAI, Deepgram, Mistral, ElevenLabs, સ્વ-હોસ્ટેડ સર્વર, અથવા અન્ય સંકુચિત સર્વર સાથે કનેક્ટ કરો. કી macOS Keychain માં રહે છે અને ઑડિયો તમે પસંદ કરો છો તે પ્રદાતાને સીધો જાય છે.
