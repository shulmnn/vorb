Vorb は、音声と音声をテキストに変換する Mac のネイティブディクテーション アプリです。キーボード ショートカットを 1 つ押して自然に話すと、どこに書いても音声入力がクリーン テキストに変換されます。

タイピングを必要としない音声入力
Option–Space を押して開始および停止するか、プッシュ トゥ トークを選択してショートカットを押したまま話します。 Vorb は、録音を文字起こしし、結果をクリップボードにコピーし、メモ、メッセージ、ドキュメント、電子メール、その他の Mac アプリでの書き込みフローを継続します。

プライベートオンデバイス Whisper
Local Whisper を使用すると、Mac 上で音声認識を完全に実行できます。API key は必要ありません。 Whisper モデルは、[モデルのダウンロード] をクリックした場合にのみダウンロードされ、代わりに既存の WhisperKit モデルをインポートできます。ローカル文字起こしにより、音声がデバイス上に保持されます。

独自の STT プロバイダーを導入する
Groq、OpenAI、Deepgram、Mistral、ElevenLabs、セルフホスト型 Whisper サーバー、または別の互換性のある音声入力エンドポイントに接続します。 API key は macOS Keychain のままで、音声は選択したプロバイダーに直接送信されます。

高速 Mac ディクテーション
• 設定可能なグローバルショートカット
• トグル・トゥ・スピーキングまたはプッシュ・トゥ・トーク
• 音声反応録音オーブ
• 自動クリップボード配信
• Optional ローカル転写履歴
• 自動言語検出
• Vorb アカウント、分析、広告、またはサブスクリプションなし

Vorb を使用すると、音声入力、STT の文字起こし、および Whisper のディクテーションが macOS にネイティブな感覚になります。 Local WhisperにはAppleシリコンが必要です。オンライン プロバイダーとモデルのダウンロードにはネットワーク接続が必要です。
