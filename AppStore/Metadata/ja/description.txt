Vorb は、キーボード ショートカット 1 つで音声をテキストに変換します。

Option–Space を押して開始および停止するか、押し続けて話すを選択します。 Vorb は録音を文字起こしし、結果をクリップボードにコピーします。

プライベート・ローカルささやき
API キーを使用せずに、Mac 上で音声を完全にテキストに変換します。モデルは、[モデルのダウンロード] をクリックした場合にのみダウンロードされます。既存の WhisperKit モデルをインポートすることもできます。

独自のプロバイダーを導入する
Groq、OpenAI、Deepgram、Mistral、ElevenLabs、セルフホステッド サーバー、または別の互換性のあるエンドポイントに接続します。キーは macOS Keychain に残り、音声は選択したプロバイダーに直接送信されます。

集中して作業できるように構築
• 設定可能なグローバルショートカット
• 切り替えて話すか、押し続けて話す
• 最小限の記録オーブ
• クリップボード配信
• Option のローカル トランスクリプト履歴
• 自動言語検出
• Vorb アカウント、分析、広告、またはサブスクリプションがない

Local Whisper には Apple シリコンが必要です。オンライン プロバイダーとモデルのダウンロードにはネットワーク接続が必要です。
