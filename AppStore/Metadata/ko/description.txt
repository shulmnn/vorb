Vorb는 하나의 키보드 단축키를 사용하여 음성을 텍스트로 변환합니다.

Option–Space를 눌러 시작하거나 중지하거나 길게 눌러 말하기를 선택하세요. Vorb는 녹음 내용을 기록하고 결과를 클립보드에 복사합니다.

개인 · 지역 속삭임
API 키 없이 Mac에서 음성을 텍스트로 완전히 녹음하세요. 모델 다운로드를 클릭한 경우에만 모델이 다운로드됩니다. 기존 WhisperKit 모델을 가져올 수도 있습니다.

자신의 서비스 제공자를 가져오세요
Groq, OpenAI, Deepgram, Mistral, ElevenLabs, 자체 호스팅 서버 또는 다른 호환 가능한 엔드포인트를 연결하세요. 키는 macOS Keychain에 유지되며 오디오는 선택한 공급자에게 직접 전달됩니다.

집중적인 작업을 위해 제작됨
• 구성 가능한 전역 단축키
• 말하기 전환 또는 말하기 대기
• 최소한의 기록 구
• 클립보드 전달
• Optional 로컬 성적표 기록
• 자동 언어 감지
• Vorb 계정, 분석, 광고 또는 구독이 없습니다.

Local Whisper에는 Apple 실리콘이 필요합니다. 온라인 공급자 및 모델 다운로드에는 네트워크 연결이 필요합니다.
