Vorb는 음성과 음성을 텍스트로 변환하는 Mac용 기본 받아쓰기 앱입니다. 키보드 단축키 하나를 누르고 자연스럽게 말하고 글을 쓸 때마다 음성 입력을 깔끔한 텍스트로 바꿔보세요.

타이핑 없이 음성 입력
Option–Space를 눌러 시작하거나 중지하거나, 눌러서 말하기를 선택하고 말하는 동안 단축키를 길게 누르세요. Vorb는 녹음 내용을 기록하고 결과를 클립보드에 복사하며 메모, 메시지, 문서, 이메일 및 기타 Mac 앱에서 글쓰기 흐름을 계속 유지합니다.

프라이빗 온디바이스 Whisper
Local Whisper를 사용하여 Mac에서 음성 인식을 완전히 실행하세요. API key는 필요하지 않습니다. Whisper 모델은 모델 다운로드를 클릭해야만 다운로드되며, 대신 기존 WhisperKit 모델을 가져올 수 있습니다. 로컬 전사는 오디오를 장치에 유지합니다.

나만의 STT 공급자 가져오기
Groq, OpenAI, Deepgram, Mistral, ElevenLabs, 자체 호스팅 Whisper 서버 또는 다른 호환 가능한 음성-텍스트 끝점을 연결합니다. API key는 macOS Keychain에 유지되며 오디오는 선택한 공급자에게 직접 전달됩니다.

빠른 Mac 받아쓰기
• 구성 가능한 전역 단축키
• 말하기 전환 또는 눌러서 말하기
• 음성 반응형 녹음 구체
• 자동 클립보드 전달
• Optional 로컬 성적표 기록
• 자동 언어 감지
• Vorb 계정, 분석, 광고 또는 구독이 없습니다.

Vorb를 사용하면 음성 입력, STT 전사 및 Whisper 받아쓰기가 macOS 고유의 느낌을 줍니다. Local Whisper에는 Apple 실리콘이 필요합니다. 온라인 공급자 및 모델 다운로드에는 네트워크 연결이 필요합니다.
