Vorb എന്നത് Mac നായുള്ള ഒരു നേറ്റീവ് ഡിക്റ്റേഷൻ ആപ്പാണ്, അത് സംഭാഷണത്തെയും ശബ്ദത്തെയും വാചകമാക്കി മാറ്റുന്നു. ഒരു കീബോർഡ് കുറുക്കുവഴി അമർത്തുക, സ്വാഭാവികമായി സംസാരിക്കുക, നിങ്ങൾ എഴുതുന്നിടത്തെല്ലാം നിങ്ങളുടെ വോയ്‌സ് ഇൻപുട്ട് ക്ലീൻ ടെക്‌സ്‌റ്റാക്കി മാറ്റുക.

വോയ്‌സ് ഇൻപുട്ട്, ടൈപ്പിംഗ് ഇല്ലാതെ
ആരംഭിക്കുന്നതിനും നിർത്തുന്നതിനും Option–Space അമർത്തുക അല്ലെങ്കിൽ സംസാരിക്കുമ്പോൾ പുഷ്-ടു-ടോക്ക് തിരഞ്ഞെടുത്ത് കുറുക്കുവഴി പിടിക്കുക. Vorb നിങ്ങളുടെ റെക്കോർഡിംഗ് ട്രാൻസ്‌ക്രൈബുചെയ്യുന്നു, ഫലം ക്ലിപ്പ്ബോർഡിലേക്ക് പകർത്തുന്നു, കുറിപ്പുകൾ, സന്ദേശങ്ങൾ, പ്രമാണങ്ങൾ, ഇമെയിൽ എന്നിവയിലും മറ്റേതെങ്കിലും Mac ആപ്പിലും നിങ്ങളുടെ എഴുത്ത് ഒഴുക്ക് നിലനിർത്തുന്നു.

ഉപകരണത്തിൽ സ്വകാര്യ Whisper
Local Whisper-നൊപ്പം നിങ്ങളുടെ Mac-ൽ സംഭാഷണം തിരിച്ചറിയൽ പൂർണ്ണമായും പ്രവർത്തിപ്പിക്കുക-API key ആവശ്യമില്ല. നിങ്ങൾ ഡൗൺലോഡ് മോഡൽ ക്ലിക്ക് ചെയ്യുമ്പോൾ മാത്രമേ Whisper മോഡലുകൾ ഡൗൺലോഡ് ചെയ്യൂ, പകരം നിങ്ങൾക്ക് നിലവിലുള്ള ഒരു WhisperKit മോഡൽ ഇറക്കുമതി ചെയ്യാൻ കഴിയും. പ്രാദേശിക ട്രാൻസ്ക്രിപ്ഷൻ നിങ്ങളുടെ ഓഡിയോ ഉപകരണത്തിൽ സൂക്ഷിക്കുന്നു.

നിങ്ങളുടെ സ്വന്തം STT ദാതാവിനെ കൊണ്ടുവരിക
Groq, OpenAI, Deepgram, Mistral, ElevenLabs, സ്വയം-ഹോസ്‌റ്റ് ചെയ്‌ത Whisper സെർവർ അല്ലെങ്കിൽ മറ്റൊരു അനുയോജ്യമായ സംഭാഷണ-ടു-ടെക്‌സ്‌റ്റ് എൻഡ്‌പോയിൻ്റ് കണക്റ്റുചെയ്യുക. നിങ്ങളുടെ API key-കൾ macOS Keychain-ൽ തുടരുന്നു, ഓഡിയോ നേരിട്ട് നിങ്ങൾ തിരഞ്ഞെടുത്ത ദാതാവിലേക്ക് പോകുന്നു.

ഫാസ്റ്റ് Mac ഡിക്റ്റേഷൻ
• ക്രമീകരിക്കാവുന്ന ആഗോള കുറുക്കുവഴി
• ടോഗിൾ-ടു-സ്പീക്ക് അല്ലെങ്കിൽ പുഷ്-ടു-ടോക്ക്
• വോയ്സ്-റിയാക്ടീവ് റെക്കോർഡിംഗ് ഓർബ്
• സ്വയമേവയുള്ള ക്ലിപ്പ്ബോർഡ് ഡെലിവറി
• Optional പ്രാദേശിക ട്രാൻസ്ക്രിപ്റ്റ് ചരിത്രം
• സ്വയമേവയുള്ള ഭാഷ കണ്ടെത്തൽ
• Vorb അക്കൗണ്ട്, അനലിറ്റിക്സ്, പരസ്യം ചെയ്യൽ അല്ലെങ്കിൽ സബ്സ്ക്രിപ്ഷൻ ഇല്ല

Vorb വോയ്‌സ് ടൈപ്പിംഗ്, STT ട്രാൻസ്‌ക്രിപ്ഷൻ, Whisper ഡിക്റ്റേഷൻ എന്നിവ macOS-ൻ്റെ നേറ്റീവ് ആണെന്ന് തോന്നിപ്പിക്കുന്നു. Local Whisper-ന് ആപ്പിൾ സിലിക്കൺ ആവശ്യമാണ്. ഓൺലൈൻ ദാതാക്കൾക്കും മോഡൽ ഡൗൺലോഡുകൾക്കും ഒരു നെറ്റ്‌വർക്ക് കണക്ഷൻ ആവശ്യമാണ്.
