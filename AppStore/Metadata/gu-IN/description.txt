Vorb Mac માટે મૂળ શ્રુતલેખન એપ્લિકેશન છે જે વાણી અને અવાજને ટેક્સ્ટમાં ફેરવે છે. એક કીબોર્ડ શોર્ટકટ દબાવો, કુદરતી રીતે બોલો અને તમે જ્યાં પણ લખો ત્યાં તમારા વૉઇસ ઇનપુટને સ્વચ્છ ટેક્સ્ટમાં ફેરવો.

વૉઇસ ઇનપુટ, ટાઇપિંગ વિના
શરૂ કરવા અને રોકવા માટે Option–Space દબાવો અથવા બોલતી વખતે પુશ-ટુ-ટોક પસંદ કરો અને શોર્ટકટ પકડી રાખો. Vorb તમારા રેકોર્ડિંગને ટ્રાંસ્ક્રાઇબ કરે છે, પરિણામને ક્લિપબોર્ડ પર કૉપિ કરે છે અને તમારા લેખન પ્રવાહને નોંધો, સંદેશાઓ, દસ્તાવેજો, ઇમેઇલ અને અન્ય કોઈપણ Mac એપ્લિકેશનમાં ચાલુ રાખે છે.

ઉપકરણ પર ખાનગી Whisper
Local Whisper સાથે તમારા Mac પર સંપૂર્ણ રીતે વાણી ઓળખ ચલાવો — API key જરૂરી નથી. જ્યારે તમે મોડલ ડાઉનલોડ કરો પર ક્લિક કરો ત્યારે જ Whisper મૉડલ ડાઉનલોડ થાય છે અને તમે તેના બદલે હાલનું WhisperKit મૉડલ આયાત કરી શકો છો. સ્થાનિક ટ્રાન્સક્રિપ્શન તમારા ઑડિયોને ઉપકરણ પર રાખે છે.

તમારા પોતાના STT પ્રદાતા લાવો
Groq, OpenAI, Deepgram, Mistral, ElevenLabs, સ્વ-હોસ્ટેડ Whisper સર્વર અથવા અન્ય સુસંગત સ્પીચ-ટુ-ટેક્સ્ટ એન્ડપોઇન્ટને કનેક્ટ કરો. તમારા API key macOS Keychain માં રહે છે અને ઑડિયો તમે પસંદ કરો છો તે પ્રદાતાને સીધો જાય છે.

ઝડપી Mac શ્રુતલેખન
• રૂપરેખાંકિત વૈશ્વિક શૉર્ટકટ
• બોલવા માટે ટૉગલ કરો અથવા વાત કરવા માટે દબાણ કરો
• વૉઇસ-રિએક્ટિવ રેકોર્ડિંગ ઓર્બ
• ઓટોમેટિક ક્લિપબોર્ડ ડિલિવરી
• Optional સ્થાનિક ટ્રાન્સક્રિપ્ટ ઇતિહાસ
• સ્વચાલિત ભાષા શોધ
• કોઈ Vorb એકાઉન્ટ, એનાલિટિક્સ, જાહેરાત અથવા સબ્સ્ક્રિપ્શન નથી

Vorb વૉઇસ ટાઇપિંગ, STT ટ્રાન્સક્રિપ્શન અને Whisper શ્રુતલેખન macOS ને મૂળ લાગે છે. Local Whisper ને Apple સિલિકોનની જરૂર છે. ઑનલાઇન પ્રદાતાઓ અને મોડલ ડાઉનલોડ માટે નેટવર્ક કનેક્શનની જરૂર છે.
