Vorb ಎಂಬುದು Mac ಗಾಗಿ ಸ್ಥಳೀಯ ಡಿಕ್ಟೇಶನ್ ಅಪ್ಲಿಕೇಶನ್ ಆಗಿದ್ದು ಅದು ಭಾಷಣ ಮತ್ತು ಧ್ವನಿಯನ್ನು ಪಠ್ಯವಾಗಿ ಪರಿವರ್ತಿಸುತ್ತದೆ. ಒಂದು ಕೀಬೋರ್ಡ್ ಶಾರ್ಟ್‌ಕಟ್ ಅನ್ನು ಒತ್ತಿ, ಸ್ವಾಭಾವಿಕವಾಗಿ ಮಾತನಾಡಿ ಮತ್ತು ನೀವು ಎಲ್ಲಿ ಬರೆದರೂ ನಿಮ್ಮ ಧ್ವನಿ ಇನ್‌ಪುಟ್ ಅನ್ನು ಕ್ಲೀನ್ ಪಠ್ಯವಾಗಿ ಪರಿವರ್ತಿಸಿ.

ಧ್ವನಿ ಇನ್ಪುಟ್, ಟೈಪಿಂಗ್ ಇಲ್ಲದೆ
ಪ್ರಾರಂಭಿಸಲು ಮತ್ತು ನಿಲ್ಲಿಸಲು Option–Space ಒತ್ತಿರಿ ಅಥವಾ ಮಾತನಾಡಲು ಪುಶ್-ಟು-ಟಾಕ್ ಆಯ್ಕೆಮಾಡಿ ಮತ್ತು ಮಾತನಾಡುವಾಗ ಶಾರ್ಟ್‌ಕಟ್ ಹಿಡಿದುಕೊಳ್ಳಿ. Vorb ನಿಮ್ಮ ರೆಕಾರ್ಡಿಂಗ್ ಅನ್ನು ಲಿಪ್ಯಂತರ ಮಾಡುತ್ತದೆ, ಫಲಿತಾಂಶವನ್ನು ಕ್ಲಿಪ್‌ಬೋರ್ಡ್‌ಗೆ ನಕಲಿಸುತ್ತದೆ ಮತ್ತು ನಿಮ್ಮ ಬರವಣಿಗೆಯ ಹರಿವನ್ನು ಟಿಪ್ಪಣಿಗಳು, ಸಂದೇಶಗಳು, ಡಾಕ್ಯುಮೆಂಟ್‌ಗಳು, ಇಮೇಲ್ ಮತ್ತು ಯಾವುದೇ ಇತರ Mac ಅಪ್ಲಿಕೇಶನ್‌ನಲ್ಲಿ ಚಲಿಸುವಂತೆ ಮಾಡುತ್ತದೆ.

ಖಾಸಗಿ ಆನ್-ಡಿವೈಸ್ Whisper
Local Whisper ನೊಂದಿಗೆ ನಿಮ್ಮ Mac ನಲ್ಲಿ ಸಂಪೂರ್ಣವಾಗಿ ಭಾಷಣ ಗುರುತಿಸುವಿಕೆಯನ್ನು ರನ್ ಮಾಡಿ-ಯಾವುದೇ API key ಅಗತ್ಯವಿಲ್ಲ. ನೀವು ಡೌನ್‌ಲೋಡ್ ಮಾಡೆಲ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿದಾಗ ಮಾತ್ರ Whisper ಮಾದರಿಗಳು ಡೌನ್‌ಲೋಡ್ ಆಗುತ್ತವೆ ಮತ್ತು ಬದಲಿಗೆ ನೀವು ಅಸ್ತಿತ್ವದಲ್ಲಿರುವ WhisperKit ಮಾದರಿಯನ್ನು ಆಮದು ಮಾಡಿಕೊಳ್ಳಬಹುದು. ಸ್ಥಳೀಯ ಪ್ರತಿಲೇಖನವು ನಿಮ್ಮ ಆಡಿಯೊವನ್ನು ಸಾಧನದಲ್ಲಿ ಇರಿಸುತ್ತದೆ.

ನಿಮ್ಮ ಸ್ವಂತ STT ಪೂರೈಕೆದಾರರನ್ನು ತನ್ನಿ
Groq, OpenAI, Deepgram, Mistral, ElevenLabs, ಸ್ವಯಂ-ಹೋಸ್ಟ್ ಮಾಡಿದ Whisper ಸರ್ವರ್ ಅಥವಾ ಇನ್ನೊಂದು ಹೊಂದಾಣಿಕೆಯ ಭಾಷಣದಿಂದ ಪಠ್ಯದ ಅಂತ್ಯಬಿಂದುವನ್ನು ಸಂಪರ್ಕಿಸಿ. ನಿಮ್ಮ API key ಗಳು macOS Keychain ನಲ್ಲಿ ಉಳಿಯುತ್ತವೆ ಮತ್ತು ಆಡಿಯೋ ನೇರವಾಗಿ ನೀವು ಆಯ್ಕೆ ಮಾಡಿದ ಪೂರೈಕೆದಾರರಿಗೆ ಹೋಗುತ್ತದೆ.

ವೇಗದ Mac ಡಿಕ್ಟೇಶನ್
• ಕಾನ್ಫಿಗರ್ ಮಾಡಬಹುದಾದ ಜಾಗತಿಕ ಶಾರ್ಟ್‌ಕಟ್
• ಟಾಗಲ್-ಟು-ಸ್ಪೀಕ್ ಅಥವಾ ಪುಶ್-ಟು-ಟಾಕ್
• ಧ್ವನಿ-ಪ್ರತಿಕ್ರಿಯಾತ್ಮಕ ರೆಕಾರ್ಡಿಂಗ್ ಮಂಡಲ
• ಸ್ವಯಂಚಾಲಿತ ಕ್ಲಿಪ್‌ಬೋರ್ಡ್ ವಿತರಣೆ

• ಸ್ವಯಂಚಾಲಿತ ಭಾಷೆ ಪತ್ತೆ
• Vorb ಖಾತೆ, ವಿಶ್ಲೇಷಣೆ, ಜಾಹೀರಾತು ಅಥವಾ ಚಂದಾದಾರಿಕೆ ಇಲ್ಲ

Vorb ಧ್ವನಿ ಟೈಪಿಂಗ್, STT ಪ್ರತಿಲೇಖನ ಮತ್ತು Whisper ಡಿಕ್ಟೇಶನ್ ಅನ್ನು macOS ಗೆ ಸ್ಥಳೀಯವಾಗಿ ಭಾವಿಸುವಂತೆ ಮಾಡುತ್ತದೆ. Local Whisper ಗೆ Apple ಸಿಲಿಕಾನ್ ಅಗತ್ಯವಿದೆ. ಆನ್‌ಲೈನ್ ಪೂರೈಕೆದಾರರು ಮತ್ತು ಮಾದರಿ ಡೌನ್‌ಲೋಡ್‌ಗಳಿಗೆ ನೆಟ್‌ವರ್ಕ್ ಸಂಪರ್ಕದ ಅಗತ್ಯವಿದೆ.
