Vorb হল Mac-এর জন্য একটি নেটিভ ডিক্টেশন অ্যাপ যা বক্তৃতা এবং ভয়েসকে পাঠ্যে পরিণত করে। একটি কীবোর্ড শর্টকাট টিপুন, স্বাভাবিকভাবে কথা বলুন এবং আপনি যেখানেই লিখুন সেখানে আপনার ভয়েস ইনপুটকে পরিষ্কার পাঠে পরিণত করুন।

ভয়েস ইনপুট, টাইপিং ছাড়া
শুরু করতে এবং থামাতে Option–Space টিপুন, অথবা কথা বলার সময় পুশ-টু-টক বেছে নিন এবং শর্টকাট ধরে রাখুন। Vorb আপনার রেকর্ডিং প্রতিলিপি করে, ফলাফল ক্লিপবোর্ডে অনুলিপি করে এবং নোট, বার্তা, নথি, ইমেল এবং অন্য যেকোন Mac অ্যাপে আপনার লেখার প্রবাহকে সচল রাখে।

ডিভাইসে ব্যক্তিগত Whisper
Local Whisper-এর সাথে আপনার Mac-এ সম্পূর্ণরূপে বক্তৃতা শনাক্তকরণ চালান—কোনও API key প্রয়োজন নেই৷ Whisper মডেলগুলি তখনই ডাউনলোড হয় যখন আপনি মডেল ডাউনলোড করুন ক্লিক করুন এবং আপনি পরিবর্তে একটি বিদ্যমান WhisperKit মডেল আমদানি করতে পারেন৷ স্থানীয় ট্রান্সক্রিপশন আপনার অডিও ডিভাইসে রাখে।

আপনার নিজস্ব STT প্রদানকারী আনুন
Groq, OpenAI, Deepgram, Mistral, ElevenLabs, একটি স্ব-হোস্টেড Whisper সার্ভার, বা অন্য একটি সামঞ্জস্যপূর্ণ স্পিচ-টু-টেক্সট এন্ডপয়েন্ট সংযোগ করুন। আপনার API keyগুলি macOS Keychain-এ থাকে এবং অডিও সরাসরি আপনার নির্বাচিত প্রদানকারীর কাছে যায়৷

দ্রুত Mac ডিকটেশন
• কনফিগারযোগ্য গ্লোবাল শর্টকাট
• টগল-টু-স্পিক বা পুশ-টু-টক
• ভয়েস-প্রতিক্রিয়াশীল রেকর্ডিং কক্ষ
• স্বয়ংক্রিয় ক্লিপবোর্ড ডেলিভারি
• Optional স্থানীয় প্রতিলিপি ইতিহাস
• স্বয়ংক্রিয় ভাষা সনাক্তকরণ
• কোন Vorb অ্যাকাউন্ট, বিশ্লেষণ, বিজ্ঞাপন, বা সদস্যতা নেই৷

Vorb ভয়েস টাইপিং, STT ট্রান্সক্রিপশন, এবং Whisper শ্রুতিলেখাকে macOS-এর নেটিভ মনে করে। Local Whisper অ্যাপল সিলিকন প্রয়োজন। অনলাইন প্রদানকারী এবং মডেল ডাউনলোডের জন্য একটি নেটওয়ার্ক সংযোগ প্রয়োজন।
