Support

Questions, bug reports, and feature ideas are welcome.

Email support Open a GitHub issue

When reporting a problem

Include your macOS version, Mac model, selected transcription provider, and the steps that reproduce the issue. Never include API keys, private recordings, or sensitive transcript text.

Local Whisper

Choose Local Whisper in Transcription settings, select a model, and click Download Model. Vorb never downloads a model automatically on the first recording.

Mac App Store build

The Mac App Store build copies transcripts to the clipboard. Automatic cross-app paste is available only in the separately distributed direct build because App Sandbox does not permit that automation.
